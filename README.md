# User manual

Step 0. Following libraries are needed to run the shiny app: _Suerat_, _SeuratData_, _sp_, _shinytitle_.

Step 1: Download/clone the package from the Seurat_Xplorer repository.

Step 2. Run the app via the _app.R_ file.

Step 3: Upload Seurat .RDS file.

Step 4: Select features/genes to visualize and the _group.by_ and/or _split.by_ parameters where applicable.

Step 5: Navigate through the different tabs for the different visualizations.
